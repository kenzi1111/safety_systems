# アプリ（Care Connect）の概要
一人暮らしかつ要支援者（今後介護が必要になる可能性が高い方）向けの安否確認用チャットアプリケーションです。
管理者（行政の担当者や医師）が対象者にメッセージを送信し、48時間以上返信or既読ボタンのクリックアクションがない場合に
未返信ユーザリスト一覧にユーザ情報が登録されます。
レスポンシブ対応しているためスマホからの確認も可能です。
